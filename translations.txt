# Mechanism
howLabel = Mechanism
howH     = How does the <em>index</em> work?
h1t = Monthly CPI measurement
h1b = STATEC tracks ~7,700 prices across ~580 outlets each month. Since 2018, supermarket scanner data (~90,000 additional varieties) supplements the survey.
h2t = Linking to the 1948 base
h2b = The national CPI (IPCN) is multiplied by a constant linking factor to obtain the index base 100 = 01.01.1948. This factor changes only when law neutralises specific prices (tobacco excise, CO₂ tax…).
h3t = Six-month moving average
h3b = The arithmetic mean of the last 6 linked indices is computed monthly. When it exceeds the trigger threshold (= last rate × 1.025), a new tranche takes effect the following month.
h4t = Automatic 2.5% raise
h4b = All wages, salaries and pensions rise by 2.5% from the date of publication. Mandatory public order (Art. L.223-1 Labour Code) — no exceptions whatsoever.

# FAQ

faqH     = Everything about the <em>index</em>
faqs.1.q = When is the next wage index tranche?
faqs.1.a = According to STATEC forecasts of 9 February 2026, the next tranche is expected in Q2 2026 (central scenario, inflation 1.8%). The trigger threshold is 992.24 points.
faqs.2.q = How much does each tranche increase salaries?
faqs.2.a = Each tranche triggers an automatic 2.5% increase in all wages, salaries and pensions, with no exceptions permitted.
faqs.3.q = What is the current application rate?
faqs.3.a = The current application rate is 968.04 points (in force since 1 May 2025). The next trigger threshold is 992.24 points.
faqs.4.q = Does indexation apply to all employees?
faqs.4.a = Yes. The mechanism is mandatory public order and applies to all private and public sector employers. No downward derogations are possible, even by collective agreement.
faqs.5.q = How is the six-month moving average calculated?
faqs.5.a = It is the arithmetic mean of the last 6 linked indices (base 01.01.1948). When it exceeds the trigger threshold, a new tranche takes effect the following month.
faqs.6.q = Are bonuses also indexed?
faqs.6.a = Only remuneration elements explicitly indexed in the employment contract or collective agreement are automatically adjusted. Non-indexed bonuses remain unchanged.
